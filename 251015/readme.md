🧮 주요 코어 객체
📆 Date 객체
- 시간 정보 저장 및 처리
- 주요 메소드: getDate(), getHours(), toLocaleString()
📚 String 객체
- 문자열 처리용 객체
- 주요 메소드: charAt(), concat(), indexOf(), slice(), split(), replace(), trim()
📊 Array 객체
- 여러 데이터를 순차적으로 저장
- 생성 방법: [] 또는 new Array()
- 주요 메소드: concat(), join(), reverse(), slice(), sort(), toString()
- length 프로퍼티로 배열 크기 확인 및 조절 가능
📐 Math 객체
- 수학 계산용 객체
- 주요 메소드: Math.sqrt(), Math.PI, Math.random(), Math.floor()
