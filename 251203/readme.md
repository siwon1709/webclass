
얼굴 분석 기반 음악 추천 웹사이트 소개 레포트
1. 개요
본 웹사이트는 사용자가 얼굴 사진을 업로드하면 인공지능 기술을 활용해 감정을 분석하고, 그 결과에 맞는 음악을 추천해주는 서비스입니다. 특히 Spotify와 연동하여 실제 음악을 바로 재생할 수 있도록 구현되어 있어, 감정과 음악을 연결하는 새로운 사용자 경험을 제공합니다.

2. 주요 기능
사진 업로드 기능: 사용자가 자신의 얼굴 사진을 업로드하면 시스템이 이를 인식합니다.

감정 분석: 얼굴 인식 알고리즘(face-api.js 등)을 활용하여 표정에서 감정을 추출합니다.

음악 추천: 분석된 감정에 따라 Spotify API를 통해 적합한 음악을 추천합니다.

즉시 재생: 추천된 음악은 웹사이트 내에서 바로 재생할 수 있어 편리합니다.

3. 장점
직관적인 흐름: 사진 업로드 → 감정 분석 → 음악 추천이라는 단순하고 명확한 사용자 여정.

실시간 반응성: 감정 분석 결과가 즉시 반영되어 음악 추천으로 이어짐.

개인화 경험: 사용자의 현재 감정 상태에 맞춘 맞춤형 음악 제공.

확장 가능성: Spotify API 연동을 통해 다양한 음악 데이터베이스 활용 가능.

4. 개선 방향
UI/UX 개선: 업로드 버튼과 결과 표시를 더 직관적으로 배치하고, 감정 결과를 색상·아이콘으로 시각화.

실시간 카메라 지원: 사진 업로드뿐 아니라 웹캠을 통한 실시간 감정 분석 기능 추가.

개인화 기록: 사용자의 감정 분석 결과와 추천 음악을 히스토리로 저장하여 맞춤 플레이리스트 제공.

소셜 공유 기능: 감정과 추천 음악을 SNS에 공유할 수 있는 기능 도입.

다중 콘텐츠 추천: 음악뿐 아니라 감정에 맞는 글귀, 이미지, 영화 등 다양한 콘텐츠 추천.

5. 결론
본 웹사이트는 감정 분석과 음악 추천을 결합한 혁신적인 서비스로, 사용자에게 새로운 형태의 음악 경험을 제공합니다. 현재는 기본적인 기능 구현에 집중되어 있으나, 향후 UI/UX 개선과 개인화·소셜 기능 확장을 통해 더욱 매력적인 플랫폼으로 발전할 수 있습니다.
