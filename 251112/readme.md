
이 사이트는 여행자를 위한 맛집 및 관광명소 추천 플랫폼입니다. 주요 기능은 다음과 같습니다:

도시, 카테고리, 키워드로 장소 검색 맛집, 관광명소, 카페, 야경/포인트, 가성비 등 다양한 필터 인기순, 평점순, 최신순 정렬 즐겨찾기(찜) 기능 장소 제안 폼을 통한 사용자 참여 다크 모드 지원 장소별 상세 정보와 지도 보기 모든 데이터는 사용자의 브라우저(로컬)에 저장되는 데모 형태로, 회원가입이나 서버 저장 없이 간편하게 사용할 수 있습니다.
