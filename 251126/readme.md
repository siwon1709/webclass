AI 기반 음악 Mood Player
사용 API: Spotify API, Emotion Detection API

아이디어: 사용자가 얼굴 사진을 업로드하면 감정을 분석해 그에 맞는 음악을 자동 재생. 예시: 웃는 얼굴 → 신나는 음악, 피곤한 얼굴 → 잔잔한 음악 등등

📌 프로젝트 개요
프로젝트명: AI 기반 음악 Mood Player

핵심 아이디어: 사용자가 얼굴 사진을 업로드하면 감정 분석을 통해 현재 기분에 맞는 음악을 자동 추천 및 재생

사용 API:

Spotify API → 음악 검색 및 재생

Emotion Detection API → 얼굴 이미지 기반 감정 인식

🧠 동작 흐름
사용자 입력: 얼굴 사진 업로드

감정 분석: Emotion Detection API로 사진 속 표정을 분석

웃는 얼굴 → 행복/즐거움

찡그린 얼굴 → 슬픔/우울

피곤한 얼굴 → 무기력/휴식 필요

음악 매칭: 감정 결과를 기반으로 Spotify API에서 적합한 플레이리스트 또는 곡 검색

자동 재생: 분석된 감정에 맞는 음악을 사용자에게 바로 재생

💡 기대 효과
개인화된 음악 경험: 사용자의 현재 감정에 맞는 음악 제공

감정 관리: 음악을 통해 기분 전환 및 스트레스 완화

새로운 음악 발견: 평소 듣지 않던 장르도 감정 기반 추천으로 접할 수 있음

🚀 확장 가능성
실시간 감정 분석: 웹캠을 통한 실시간 감정 변화에 맞춘 음악 자동 변경

웨어러블 연동: 스마트워치의 심박수·피로도 데이터와 결합해 더 정밀한 추천

소셜 기능: 친구와 감정 기반 플레이리스트 공유
